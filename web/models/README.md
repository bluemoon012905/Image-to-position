# TF.js Model Drop Folder

Place converted TensorFlow.js model files here, e.g.:

- `web/models/stone_classifier/model.json`
- `web/models/stone_classifier/group1-shard1of1.bin`

Then in the UI Step 2:

- set Model URL to `./models/stone_classifier/model.json`
- click `Load ML model`
- click `Extract stones`
